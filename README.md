# English A1 Trainer MVP

A static Duolingo-like MVP for Ukrainian speakers learning English A1. It uses Vite, React, TypeScript, React Router HashRouter, static JSON content, and localStorage.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is generated in `dist`.

## GitHub Pages

Deploy the `dist` folder to GitHub Pages. The app uses `HashRouter`, so routes such as `#/course/english-a1` work on static hosting without server rewrites.
